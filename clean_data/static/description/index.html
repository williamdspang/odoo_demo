Mass Clear Data(Clean data)

Main Features:
This module helps the user clear unwanted testing data using wizard on single click of button.

How to Use:
User need to activate the developer mode to access the "Clean Data" Menu in the Settings
Steps to activate developer mode:
Navigate to: Settings > General Settings > Developer Tools.

Clean Data Menu

Once the Developer Mode is activated, a new menu named "Clean Data" will visible.
Navigate to: Settings > Clean Data > Clean Data.

Available Options for Data Cleaning

Easily clear following object / tables data using the checkboxes:
1.Sales & All Transfers
2.Purchase & All Transfers
3.Only Transfers
4.All Invoicing, Payments & Journal Entries
5.Only Journal Entries
6.Customers & Vendors
7.Chart Of Accounts & All Accounting Data
User can also clean all above object using single click by checking "All Data" boolean.

Data will be deleted only if the table exist in the database. For example, If Purchase module is not installed, Purchase checkbox will be visible on the popup but as the table doesn't exist it will have no effect.

Notes: Do not use this module in production environment.
Data is deleted via SQL query and you will not be able to restore it.
This module supports only core objects/tables of the odoo and not custom objects/tables.
